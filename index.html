Spam Detection Program

As mentioned in the project above, my Software Development and Integreation course required me to make a spam detection program. We were given a data set of emails that were labeled spam or ham(not spam). Using a metioned involing finding the probability of certain word found in alot of spam emailing and putting it in a hashmap, we were able to detect spam emails with a 89% accuracy. The project used Java in the backend and JS/HTML/CSS for the frontend display. The project taught me a lot about data reading, as it gave me the freedom to find inovative solutions for detecting spam. It was alot of fun working on the project and I'd like to explore popular technics for spam detection.

Sudoku Solver

This was a project I started in grade 11, but didn't have the proper knowledge to complete untill the end of my first universtiy semester. I tried many different technics of solving a given sudoku matrix using interation, but I wasn't able to complete it until I learned recursion in my intro cs course. I was able to use recurcive backtracking to attempt thousands of solutions for the board, going back one step every time there was an illegal move, until the board was solved. This project really pushed my coding skills and required me to try many methods over 2 years, but I am very glad I did it and I taught me so much.
